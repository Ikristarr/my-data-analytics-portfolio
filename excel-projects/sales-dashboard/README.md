# Sales Dashboard

## Overview
This Excel dashboard analyzes sales performance, product profitability, customer contribution, and profit trends across different segments.

## Tool Used
- Excel

## Project Goal
The goal of this project was to turn raw sales data into a clear visual dashboard that helps identify trends, top performers, and business opportunities.

## Dashboard Preview

![Dashboard 1](dashboard1.png)
![Dashboard 2](dashboard2.png)

## Key Insights
- Identified the top 5 most profitable products
- Highlighted which customers contributed the most profit
- Showed profit contribution by age group
- Revealed weekday and monthly profit trends
- Compared regional contribution to total profit
